# Resolume Development Documentation

  Developer-oriented documentation for **Resolume Arena / Avenue 7.x** — OSC control,
  FFGL plugin development, ISF shader authoring, and Wire patch development.

  🌐 **Live site**: https://onshworks-vj.github.io/resolume-dev-docs/

  ## Languages

  - 🇺🇸 [English](README.en.md)
  - 🇰🇷 [한국어](README.ko.md)

  ## Topics covered

  - **OSC** — Control Resolume from TouchOSC, Companion, Max/MSP, custom code
  - **FFGL** — Native C++ effects, sources, mixers
  - **ISF** — JSON-prefixed GLSL fragment shaders
  - **Wire** — Node-based patch development
